QA portfolio with manual tests, test automation, API testing, BDD, bug reports and test documentation. Este repositório reúne projetos desenvolvidos para demonstrar minhas habilidades em testes manuais, automação de testes, testes de APIs e documentação de QA. Os projetos foram criados com o objetivo de aplicar boas práticas da área e simular cenários encontrados no dia a dia de equipes de qualidade de software.

Desenvolver e executar casos de teste manuais; Elaborar planos de teste e relatórios de bugs; Automatizar testes Web utilizando Cypress e Playwright; Realizar testes em APIs REST; Implementar cenários utilizando BDD; Aplicar boas práticas de Git e GitHub; Construir um portfólio profissional de QA;
