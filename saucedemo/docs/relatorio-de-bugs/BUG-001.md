# BUG-001 - Descrição do produto exibe trecho de código ao usuário

| Campo | Descrição |
|--------|-----------|
| ID | BUG-001 |
| Módulo | Produtos |
| Severidade | Baixa |
| Prioridade | Média |
| Tipo | Interface / Conteúdo |
| Ambiente | https://www.saucedemo.com/ |
| Usuário | standard_user |

## Descrição

A descrição do produto **Sauce Labs Backpack** apresenta um trecho de código (`carry.allTheThings()`) visível ao usuário final.

Esse conteúdo técnico não agrega valor à descrição do produto e pode causar estranheza ou comprometer a experiência do usuário.

## Passos para reprodução

1. Acessar o Sauce Demo.
2. Realizar login com um usuário válido.
3. Localizar o produto **Sauce Labs Backpack** na lista de produtos.
4. Observar a descrição do produto.

## Resultado esperado

A descrição do produto deve conter apenas informações comerciais e descritivas, sem expor trechos de código ou nomenclaturas técnicas.

## Resultado obtido

A descrição inicia com o texto:

> **carry.allTheThings() with the sleek, streamlined Sly Pack that melds uncompromising style with unequaled laptop and tablet protection.**
