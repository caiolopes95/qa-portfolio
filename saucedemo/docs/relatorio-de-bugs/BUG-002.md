# BUG-002 - Título do produto exibe trecho de código ao usuário

| Campo | Descrição |
|--------|-----------|
| ID | BUG-002 |
| Módulo | Produtos |
| Severidade | Baixa |
| Prioridade | Média |
| Tipo | Interface / Conteúdo |
| Ambiente | https://www.saucedemo.com/ |
| Usuário | standard_user |

## Descrição

O título do produto apresenta um trecho com aparência de código (`Test.allTheThings()`), destoando do padrão de nomenclatura utilizado nos demais produtos da aplicação.

A presença desse conteúdo pode causar confusão ao usuário e comprometer a consistência da interface.

## Passos para reprodução

1. Acessar o Sauce Demo.
2. Realizar login com um usuário válido.
3. Localizar o produto **Test.allTheThings() T-Shirt (Red)** na lista de produtos.

## Resultado esperado

O título do produto deve apresentar apenas o nome comercial do item, mantendo o mesmo padrão de nomenclatura dos demais produtos.

## Resultado obtido

O título do produto é exibido como:

> **Test.allTheThings() T-Shirt (Red)**
